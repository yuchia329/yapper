- **Stage count:** the pipeline _engine_ is **12 stages**; the architecture explainer page says **13** (5 front + 8 back from the UI's view). Use 12 for the engine, 13 only when citing that page — don't mix them in one bullet.
- **LLM provider:** shipped default is **MiniMax-M3** via an OpenAI-compatible client (the README's “Claude Opus 4.8” is stale). The client is provider-pluggable (MiniMax / self-hosted vLLM / any OpenAI-compatible gateway) by config — frame it that way; it's a genuine strength.
